# 📖 Description

Smart Auto-Ecole est une application web permettant de gérer efficacement une auto-école.

Elle centralise toutes les informations relatives aux candidats, aux moniteurs, aux contrats, aux véhicules, aux séances, aux paiements et aux examens afin de simplifier la gestion administrative et pédagogique.

Ce projet est réalisé dans le cadre de mon apprentissage du développement web Full Stack afin de mettre en pratique les bonnes pratiques de conception, la programmation orientée objet (POO) et l'architecture MVC.

---

# 🎯 Objectifs

- Digitaliser la gestion d'une auto-école.
- Centraliser toutes les données.
- Automatiser les tâches administratives.
- Faciliter le suivi des candidats.
- Appliquer l'architecture MVC.
- Concevoir une base de données relationnelle avec MySQL.
- Développer une application web complète de manière professionnelle.
