# 🚗 Smart Auto-Ecole

Application web de gestion d'une auto-école développée avec **PHP (MVC)**, **MySQL**, **JavaScript**, **HTML** et **CSS**.

## 📖 Description

Smart Auto-Ecole est une application web permettant de gérer efficacement une auto-école.

Elle centralise les informations relatives aux candidats, aux moniteurs, aux véhicules, aux paiements et aux examens afin de simplifier la gestion administrative et pédagogique.

Ce projet est développé dans le but de mettre en pratique les technologies étudiées en développement web Full Stack.

---

## 🎯 Objectifs

- Digitaliser la gestion d'une auto-école.
- Centraliser les données.
- Automatiser les tâches administratives.
- Améliorer le suivi des candidats.
- Appliquer l'architecture MVC en PHP.
- Concevoir une base de données relationnelle avec MySQL.
